## Download

Grab the latest release for your platform from [GitHub Releases](https://github.com/d3ara1n/Polymerium/releases).

| Platform | File |
|----------|------|
| Windows x64 | `Polymerium-win-x64.exe` |
| Linux x64 | `Polymerium-linux-x64` |
| macOS ARM | `Polymerium-osx-arm64` |

## Windows: Enable Developer Mode

Polymerium uses symbolic links to share mod files between instances. On Windows, creating symlinks requires **Developer Mode** to be enabled.

1. Open **Settings > System > For developers**
2. Toggle **Developer Mode** on
3. Confirm the prompt

This is a one-time setup. It does not affect system security — it only allows your user account to create symbolic links.

> macOS and Linux users do not need any special configuration — symlinks work out of the box.

## First Launch

When you first open Polymerium, the OOBE (Out-of-Box Experience) wizard will guide you through:

1. **Java Detection** — Polymerium scans for installed Java runtimes. You can also add custom paths later.
2. **Language Selection** — Choose between English and Chinese.
3. **Account Setup** — Sign in with a Microsoft account, or create an offline account for local play.

Once the wizard completes, you're ready to create your first instance.
