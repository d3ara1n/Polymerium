## 支持的格式

Polymerium 支持以下格式的整合包导入和导出：

| 格式 | 导入 | 导出 |
|------|:----:|:----:|
| Modrinth (.mrpack) | 是 | 是 |
| CurseForge (.zip) | 是 | 是 |
| MultiMC (.zip) | 是 | — |

## 导入整合包

1. 点击**导入**按钮或拖放整合包文件
2. Polymerium 自动检测格式
3. 解析所有依赖并创建新实例
4. 部署并开始游戏

导入的实例是完整的 Polymerium 实例——它像其他实例一样享受符号链接去重的好处。

## 导出整合包

1. 选择要导出的实例
2. 点击**导出**
3. 选择目标格式（Modrinth 或 CurseForge）
4. 选择要包含的组件（模组、配置、资源包等）
5. 点击**导出**生成文件

导出的整合包是独立的，可以与任何人分享——他们不需要 Polymerium 就能使用。

## 从其他启动器迁移

从 Prism Launcher、MultiMC 或其他启动器迁移？只需在旧启动器中以支持的格式导出整合包，然后导入到 Polymerium 中。所有模组和配置都会保留。
