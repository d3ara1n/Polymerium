## What Are Snapshots?

Snapshots let you save the complete state of your game at any point — mods, configs, saves, everything. Think of it like a save state in an emulator, but for your entire Minecraft instance.

## Creating a Snapshot

1. Select your instance
2. Click the **Snapshots** tab
3. Click **Create Snapshot**
4. Give it a name and optional description

Polymerium captures the full instance state instantly using content-addressable storage — no heavy file copying.

## Restoring a Snapshot

1. Go to the **Snapshots** tab
2. Find the snapshot you want
3. Click **Restore**

Your instance is returned to exactly the state it was in when the snapshot was taken. This is safe — the current state is preserved and you can always switch back.

## Comparing Snapshots

You can diff two snapshots to see exactly what changed:

- Added or removed mods
- Modified configuration files
- Changed settings

This is especially useful for debugging — take a snapshot when everything works, make changes, and compare if something breaks.

## When to Use Snapshots

- **Before updating mods** — easily roll back if an update breaks things
- **Before experimenting** — try risky mod combinations without fear
- **Before sharing** — capture a known-good state before exporting
- **During development** — track changes to your modpack over time

Snapshots are lightweight and fast, so don't hesitate to create them frequently.
