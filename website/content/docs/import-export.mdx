## Supported Formats

Polymerium supports importing and exporting modpacks in the following formats:

| Format | Import | Export |
|--------|:------:|:------:|
| Modrinth (.mrpack) | Yes | Yes |
| CurseForge (.zip) | Yes | Yes |
| MultiMC (.zip) | Yes | — |

## Importing a Modpack

1. Click the **Import** button or drag and drop a modpack file
2. Polymerium detects the format automatically
3. It resolves all dependencies and creates a new instance
4. Deploy and play

The imported instance is a full Polymerium instance — it benefits from symlink deduplication just like any other.

## Exporting a Modpack

1. Select the instance you want to export
2. Click **Export**
3. Choose the target format (Modrinth or CurseForge)
4. Select which components to include (mods, configs, resource packs, etc.)
5. Click **Export** to generate the file

Exported modpacks are self-contained and can be shared with anyone — they don't need Polymerium to use them.

## Migrating from Other Launchers

Coming from Prism Launcher, MultiMC, or another launcher? Simply export your modpack from the old launcher in a supported format, then import it into Polymerium. All your mods and configs carry over.
