## 下载

从 [GitHub Releases](https://github.com/d3ara1n/Polymerium/releases) 下载适用于你平台的最新版本。

| 平台 | 文件 |
|------|------|
| Windows x64 | `Polymerium-win-x64.exe` |
| Linux x64 | `Polymerium-linux-x64` |
| macOS ARM | `Polymerium-osx-arm64` |

## Windows：启用开发者模式

Polymerium 使用符号链接在实例之间共享模组文件。在 Windows 上，创建符号链接需要启用**开发者模式**。

1. 打开**设置 > 系统 > 开发者选项**
2. 开启**开发者模式**开关
3. 确认提示

这是一次性设置。它不会影响系统安全——只是允许你的用户账户创建符号链接。

> macOS 和 Linux 用户无需任何特殊配置——符号链接开箱即用。

## 首次启动

首次打开 Polymerium 时，OOBE（开箱体验）向导会引导你完成：

1. **Java 检测** — Polymerium 自动扫描已安装的 Java 运行时。你也可以稍后添加自定义路径。
2. **语言选择** — 在英文和中文之间选择。
3. **账户设置** — 使用微软账户登录，或创建离线账户进行本地游戏。

向导完成后，你就可以创建第一个实例了。
