title: 创建实例
description: 在 Polymerium 中创建你的第一个模组实例。
---

## 创建实例

1. 点击侧边栏或主区域的 **+** 按钮
2. 为实例命名
3. 选择 **Minecraft 版本**（如 1.21.4）
4. 选择**模组加载器**：
   - **Fabric** — 轻量级，更新快
   - **Quilt** — 兼容 Fabric，功能更多
   - **Forge** — 经典，庞大的模组生态
   - **NeoForge** — 现代 Forge 分支
5. 点击**创建**

你的实例以轻量级元数据文件的形式创建——此时还没有复制任何文件。

## 添加模组

1. 在侧边栏选择你的实例
2. 切换到**模组**标签页
3. 在 Polymerium 内直接浏览或搜索 **CurseForge** 和 **Modrinth**
4. 点击任意模组上的**安装**将其添加到实例

模组存储在共享缓存中。如果另一个实例已有相同的模组文件，Polymerium 只需链接到它，无需再次下载。

## 部署

在启动之前，Polymerium 需要**部署**你的实例——这会使用符号链接从共享缓存构建实际的游戏目录。

1. 点击**部署**（首次启动时会自动进行）
2. Polymerium 解析所有依赖并创建游戏目录
3. 点击**启动**开始游戏

部署速度很快，因为没有文件复制——只创建符号链接，几乎是瞬间完成。

## 管理多个实例

每个实例都是独立的：

- 不同的 Minecraft 版本
- 不同的模组加载器
- 不同的模组组合
- 不同的设置

但它们都通过符号链接共享相同的底层模组文件。添加一个已在其他实例中缓存的模组不会占用额外的磁盘空间。
