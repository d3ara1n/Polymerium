Welcome to the Polymerium documentation.

Polymerium is a next-generation Minecraft instance manager. Instead of
copying and storing thousands of mod files for each instance,
Polymerium describes your game setup as lightweight metadata and builds
the actual game files on-demand.

## Quick Start

1. **Download** the latest release for your platform
2. **Install** and run the setup wizard
3. **Configure** your first instance
