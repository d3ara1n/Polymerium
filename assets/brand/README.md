# Polymerium Brand Assets

Two interlocking rounded-diamond links — gold (`#EEA93B`) paired with a neutral counterpart.

## Files

| File | Colors | Use on |
| --- | --- | --- |
| `logo.svg` | gold + dark brown (`#3A2E1C`) | light backgrounds |
| `logo-dark.svg` | gold + warm off-white (`#E9E4D8`) | dark backgrounds |
| `logo-mono.svg` | single gold | tiny sizes (favicon, 16–32px) or single-color contexts |
| `logo-installer.svg` | amber box + dark-brown logo print | installer / package icons (`logo-installer-256.png`, `logo-installer-512.png` are raster exports) |

## Rules

- Source files carry **no background**. The consumer picks the surface per theme — do not bake one in.
- Never place `logo.svg` (dark-brown variant) directly on dark surfaces; use `logo-dark.svg`.
- Below ~40px the two-tone weave turns to mush; switch to `logo-mono.svg`.
- Do not recolor the gold link — `#EEA93B` is the brand accent and matches the site's primary color.
