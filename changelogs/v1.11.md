# Changelog v1.11

## [1.11.4] - 2026-07-17

### ✨ Highlights ✨

- Fix modpack-bundled sounds, music, and other resources not loading in-game

### Fixed

- Fix deleted instances reappearing as ghost entries on the instances page (#POLY-129)
- Fix modpack-bundled sounds, music, and other resources not loading in-game (#75, #POLY-128)

### Added

- Provide clear recovery guidance and a confirmed reset action for deployment conflicts caused by stale run folder files (#POLY-132)
- Provide a non-blocking Java runtime notification with a download action when an instance cannot start

### Changed

- Rework instance deployment to place modpack-bundled files as real files instead of symbolic links (#76, #POLY-130)
- Replace recent releases on the home page with Minecraft news that includes article links and publishing times
- Improve repository data caching to reduce repeated remote requests and keep package details responsive during transient repository slowdowns

### Removed

- Remove the blocking Java runtime download dialog when an instance cannot start
